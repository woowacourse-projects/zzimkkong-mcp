찜꽁에서 내 예약 목록을 조회하고 취소를 도와주세요.

## 흐름

1. **로그인 확인**: `get_my_info` 툴로 로그인 상태를 확인합니다.
   - 미로그인 시 "로그인이 필요합니다"라고 안내하고 종료합니다.

2. **예약 목록 조회**: `get_my_reservations` 툴로 내 예약 목록을 가져옵니다.
   - 예약이 없으면 "현재 예약이 없습니다"라고 안내합니다.
   - 예약이 있으면 다음 형식으로 목록을 보여줍니다:
     ```
     [예약 ID] 공간이름 (층)
       날짜/시간: YYYY-MM-DD HH:mm ~ HH:mm
       용도: 설명
     ```

3. **취소 여부 확인**: 취소할 예약이 있는지 물어봅니다.

4. **취소 실행**: 취소할 예약을 선택하면 `delete_reservation` 툴로 취소합니다.
   - 로그인 상태이므로 비밀번호 불필요
   - 취소 전 "정말 취소할까요?" 확인을 받습니다.

5. **결과 안내**: 취소 완료 후 취소된 예약 정보를 출력합니다.
