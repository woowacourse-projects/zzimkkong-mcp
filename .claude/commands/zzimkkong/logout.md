찜꽁 로그아웃합니다.

## 흐름

1. **로그인 상태 확인**: `get_my_info` 툴로 현재 상태를 확인합니다.
   - 로그인되어 있지 않으면: "현재 로그인된 계정이 없습니다."라고 안내하고 종료합니다.
   - 세션이 만료된 상태면: "세션이 이미 만료되어 있습니다. 로컬 인증 정보를 삭제할까요?" 를 물어봅니다.
     - 삭제 원하면 → `logout` 툴 실행 후 종료
     - 취소 → 종료

2. **로그아웃 확인**: 로그인 상태라면 현재 계정 정보를 보여주고 확인을 받습니다.
   ```
   [계정명] ([이메일]) 에서 로그아웃할까요?
   ```

3. **로그아웃 실행**: 확인 시 `logout` 툴을 실행합니다.

4. **결과 안내**: "로그아웃 완료. 다음 예약/조회 시 로그인이 필요합니다."라고 안내합니다.
